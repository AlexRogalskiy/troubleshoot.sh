Welcome.
The easiest way to get started with the Troubleshoot project is to go through a few short tutorials.
Once you've completed these tutorials, you'll be set to add custom Preflight Checks to your application and build your own Support Bundle for disconnected troubleshooting.

## Tutorials

#### [Preflight Checks](./preflight/introduction)
Start here to learn how to add custom preflight checks to your application.
[(get started)](./preflight/introduction)

#### [Support Bundle](./support-bundle/introduction)
When you have an application to support, start here to learn how to define a custom support bundle.
We'll walk through collecting data from the application and analyzing it for known errors and problems.
[(get started)](./support-bundle/introduction)

#### [Redactors]
If you need to redact some sensitive data that isn't automatically redacted, start here to learn how to write custom redactors.
